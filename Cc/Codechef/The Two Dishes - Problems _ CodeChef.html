Problem

Chef prepared two dishes yesterday. Chef had assigned the tastiness\textbf{tastiness}tastiness T1T_1T1​ and T2T_2T2​ to the first and to the second dish respectively. The tastiness of the dishes can be any integer\textbf{integer}integer between 000 and NNN (both inclusive).

Unfortunately, Chef has forgotten the values of T1T_1T1​ and T2T_2T2​ that he had assigned to the dishes. However, he remembers the sum of the tastiness of both the dishes - denoted by SSS.

Chef wonders, what can be the maximum possible absolute difference between the tastiness of the two dishes. Can you help the Chef in finding the maximum absolute difference?

It is guaranteed that at least one pair {T1,T2}\{T_1, T_2\}{T1​,T2​} exist such that T1+T2=S,0≤T1,T2≤NT_1 + T_2 = S, 0 \leq T_1, T_2 \leq NT1​+T2​=S,0≤T1​,T2​≤N.

Input Format

  • The first line of input contains a single integer TTT, denoting the number of testcases. The description of the TTT testcases follows.
  • The first and only line of each test case contains two space-separated integers NNN, SSS, denoting the maximum tastiness and the sum of tastiness of the two dishes, respectively.

Output Format

For each testcase, output a single line containing the maximum absolute difference between the tastiness of the two dishes.

Constraints

  • 1≤T≤1031 \leq T \leq 10^31≤T≤103
  • 1≤N≤1051 \leq N \leq 10^51≤N≤105
  • 1≤S≤2⋅1051 \leq S \leq 2 \cdot 10^51≤S≤2⋅105
